# 技术雷达

技术雷达是一个可视化工具，用于跟踪和展示技术趋势、工具和框架的采用状态。它将技术项目按照不同的象限和采用阶段进行分类，帮助团队做出更明智的技术决策。

## 主要特点

- **四象限布局**：技术项目按照不同领域（如工具、平台、技术和框架等）分类
- **采用阶段**：从内到外分为试验、采用、评估和暂缓四个阶段
- **数据点可视化**：每个技术项目以点的形式展示在雷达图上
- **交互式体验**：支持点击和悬停查看详细信息

## 使用方法

1. 在雷达图上查看当前技术分布
2. 点击数据点查看详细信息
3. 使用筛选功能关注特定象限或阶段
4. 定期更新雷达图以反映技术趋势变化

## 技术实现

本项目基于Vue.js构建，使用SVG进行雷达图的绘制和交互。
